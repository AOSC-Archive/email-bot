# E-mail Processor

## Installation

(You must to install Node.js and npm at first)

Install dependencies:
```sh
npm install
```

## Configuration

Create a new file `mailbox.json` on the source root. For example:

```json
{
  "user": "somebody@gmail.com",
  "password": "wwwwwwwwww",
  "host": "imap.gmail.com",
  "port": 993,
  "tls": true
}
```

## Usage

```sh
npm start
```

That's all. Too good to be true? Give it a go.
